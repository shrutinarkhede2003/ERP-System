# ERP System (Full Stack)

## 🚀 Tech Stack
- Backend: Spring Boot, Java, JWT, Hibernate
- Frontend: Angular
- Database: MySQL

## 📌 Features
- Product Management
- Customer & Supplier Management
- Sales & Purchase Orders
- GRN (Inventory Management)
- Invoice Generation (PDF)
- Dashboard & Reports
- Role-based Authentication

## 🔐 Roles
- Admin
- Sales Executive
- Purchase Manager
- Inventory Officer
- Accountant

## ▶️ How to Run

### Backend
cd backend
mvn spring-boot:run

### Frontend
cd frontend
npm install
ng serve
